# Contributing to Semptify55

Thank you for your interest in contributing to Semptify55! This is a housing justice tool built by and for tenants.

## Ways to Contribute

- **Code** - Fix bugs, add features
- **Design** - Improve mobile UX, accessibility
- **Documentation** - Write guides, improve README
- **Legal Content** - Tenant rights information, tidbits
- **Testing** - Report bugs, test on mobile devices
- **Translation** - Make it accessible in more languages

## Development Setup

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/semptify55.git`
3. Create a branch: `git checkout -b feature/my-feature`
4. Make changes
5. Test locally: `pytest`
6. Commit: `git commit -m "Add my feature"`
7. Push: `git push origin feature/my-feature`
8. Open a Pull Request

## Code Standards

- Python 3.12+
- Type hints required
- Async/await for I/O operations
- Mobile-first design
- Privacy by default
- No tracking, no ads, no data mining

## Questions?

Open an issue or discussion. We're friendly!
